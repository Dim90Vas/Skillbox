Файл libopenbgi.a нужно поместить в папку Code Blocks\MinGw\lib
Сам файл в архиве: libopenbgi.rar
Далее нужно закинуть файл graphics.h в: Code Blocks\MinGw\include
Сам файл в архиве: graphics.rar
Теперь нужно создать консольное приложение на языке C. Создали. Затем нужно подключить статическую библиотеку таким образом. Открываем вкладку: проект - опции сборки - linker settings. Под полем: библиотеки компоновщика нажимаем кнопку "добавить", откроется окно , в котором надо ввести путь к файлу нашей библиотеки или нажав кнопку справа выбрать её на компе. Затем подтверждаем выбор нажав кнопку ok.
Так как библиотека основана на использовании gdi функций, то нужно ещё подключить к проекту стандартную библиотеку gdi32.a, тем же самым образом.
